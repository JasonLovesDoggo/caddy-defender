# Security Policy

## Supported Versions

As this project is still in active development and will consistently have breaking changes pre v1.0, only the latest version will be supported for vulns.

## Reporting a Vulnerability

please send me an email @ `security@jasoncameron.dev` following the [github securitylab report template](https://github.com/github/securitylab/blob/main/docs/report-template.md#vulnerability-report)
